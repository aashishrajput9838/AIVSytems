Free Deployment Guide

Deploy your AIV Systems Chrome Extension and Backend for Free

Overview

This guide shows you how to deploy your AIV Systems Chrome Extension and backend services completely free using Firebase's generous free tier.

.Firebase Free Tier Benefits

  • Cloud Functions: 125K invocations/month
  • Firestore: 1GB storage, 50K reads/day
  • Hosting: 10GB storage, 360MB/day bandwidth
  • Authentication: 50K monthly active users
  • No credit card required

Step-by-Step Deployment

1

Create Firebase Project

  1. Go to Firebase Console
  2. Click "Create a project"
  3. Enter project name (e.g., "aiv-system")
  4. Disable Google Analytics to stay on free tier
  5. Click "Create project"
2

Install Firebase Tools

npm install -g firebase-tools
3

Login and Initialize

firebase login
firebase init

When prompted, select:

  • Firestore
  • Functions
  • Hosting
4

Deploy Your Services

firebase deploy

This will deploy all services at once. You can also deploy individually:

firebase deploy --only functions
firebase deploy --only firestore
firebase deploy --only hosting

Chrome Extension Deployment

For the Chrome Extension, you have several free options:

Option 1: Manual Installation (Always Free)

  1. Zip the chrome-extension folder
  2. Users can download and install manually:
    • Go to chrome://extensions
    • Enable "Developer mode"
    • Click "Load unpacked"
    • Select the extension folder

Option 2: GitHub Repository (Always Free)

  1. Create a GitHub repository
  2. Push your chrome-extension folder
  3. Users can download the repository and install manually
Tip: For public distribution, there's a one-time $5 fee for the Chrome Web Store Developer account. This is a one-time fee and allows you to publish extensions that anyone can install with one click.

AIV System Frontend Deployment

Deploy your React frontend using Firebase Hosting:

1

Build Your React App

npm run build
2

Deploy to Firebase Hosting

firebase deploy --only hosting

Cost Optimization Tips

  • Stay Within Free Limits: Monitor usage in Firebase Console
  • Optimize Functions: Minimize execution time and calls
  • Efficient Queries: Use indexes and limit data retrieval
  • Cache Data: Implement caching where possible
Monitoring Tip: Set up billing alerts even on the free tier to prevent accidental charges if you exceed free limits.

Scaling Beyond Free Tier

If your application grows beyond the free tier limits:

  • Firebase Pay-as-you-go is very affordable
  • Most applications stay within free limits
  • First $200 monthly credit for new Google Cloud projects
